# Guide du Pavot de Californie

## Description
Ce site web est un guide virtuel étape par étape pour apprendre à cultiver le pavot de Californie (Eschscholzia californica). Il est conçu pour les débutants en jardinage et explique toutes les étapes, de la germination à la floraison, avec des photos et des conseils pratiques.

## Contenu du site
- **index.html** : Page principale du site
- **Images** : Photos illustrant chaque étape de la culture du pavot
- **Fiches de lecture** : Sources utilisées pour rédiger le guide
- **Tableaux et schémas** : Plans de culture et organisation des étapes

## Comment ouvrir le site
1. Télécharge tous les fichiers du projet dans un même dossier.
2. Ouvre le fichier `index.html` dans un navigateur web (Chrome, Firefox, Edge, Safari).
3. Toutes les images et les tableaux seront visibles si elles sont dans le même dossier que le fichier HTML.

## Objectifs pédagogiques
- Fournir un guide clair et visuel pour cultiver le pavot de Californie.
- Permettre aux utilisateurs de suivre facilement toutes les étapes de la culture.
- Encourager les débutants à expérimenter le jardinage de manière autonome.

## Technologies utilisées
- HTML
- CSS (optionnel pour la mise en page)
- Images et illustrations pour la visualisation des étapes
